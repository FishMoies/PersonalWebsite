## 一些感想 Something I want to say
>这是一个正式的长期项目，所以会在后续进行进一步维护。  

开发该项目的起因是我准备做一个个人的作品展示集。当我做好了其他的vue作品并辛辛苦苦完成域名备案时，我发现我的主域名应该挂一个主网站；于是就有了该项目的诞生。
该项目与我其他vue项目有明显不同的是，你可以感受到这一项目更加具有叙事感，而不是单纯的产品展示或者论坛。  
最显著的特征是，这个项目刻意参考了GitHub的主页设计，用gsap库封装了一个模拟打字的标题效果。这也是我第一次用专业的动画组件，所以难免有些生疏，可能用的不是最佳方案，但是你绝对可以挂到你自己的项目上然后狠狠装一波。  
这个项目立项的时候，我写的代码特别多。因为我已经几天没去上专业课了，主要原因是准备考试的同时还要写另外两个项目，所以我的导员和我谈话，让我好好上课。这也得赖我，因为我因为心血来潮报了很多的比赛项目，导致现在有一点分身乏术了。  
当然，有些话是不能对老师讲的，毕竟我不可能真的说“我的专业课内容我已经都会了，我觉得去上课是浪费时间”。不过我确实打心眼里是这样想的。  
其实这一年来我都很焦虑，因为我发现我们这个专业的人马上要被AI替代了，所以我一直在尽可能参加更多比赛，争取校招进一个铁饭碗；但是现在想来，其实也没什么必要。假使AI真的能在三年后替代我，那么十年后的生产力也一定能养活世界上所有的人了。  
因为我的GitHub账户没有多少人看，所以我一般会把我真实的感受写在readme里面，这样以后自己找自己写的东西也能够看看以前的我在想什么。
